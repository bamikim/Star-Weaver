🌌 스타위버 (StarWeaver) 전체 기획서 v2.0


---

1. 게임 개요

장르: 방치형 수집 RPG (Idle Collection RPG)
플랫폼: 모바일 (Android / iOS)
개발: Godot 4.x / GDScript
플레이 스타일: 짧은 반복 세션 + 오프라인 성장


---

🎯 핵심 컨셉

👉 “장비를 장착하는 게임이 아니라,
정령이 스스로 싸우는 게임”

장비 = 정령

캐릭터 = 중심점

전투 = 정령이 수행



---

🎯 차별 포인트

1. WeaponSpirit 시스템 (핵심)


2. 연못 기반 가챠 → 감성 연출 중심


3. 자동 환원 → 스트레스 제거


4. 짧은 세션 루프 (1~3분)




---

2. 세계관

별빛을 엮어 세상을 유지하는 “연못”이 오염되기 시작했다.

오염은:

숲 → 항구 → 하늘 → 심연
으로 확산된다.


플레이어는 “스타위버”가 되어
정령들과 함께 세계를 정화한다.

👉 핵심 감정:

“죽이는 게 아니라 정화한다”



---

3. 핵심 게임 루프

🔁 1회 플레이 흐름 (1~3분)

시각적 피드백 강조

---

4. 이탈 유도
